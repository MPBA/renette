ReNette WebServices

This is a web-interface for network analysis. It supports inference methods, quantitative network distance and network stability analysis.

Network Distance

A quantitative analysis of differences between networks. Three methods are available, one based on an edit distance, another based on a spectral distance and a product metric combining contributions from edit and spectral distances.

Network Inference

The network inference procedure allows the user to compute his own network starting from row data. A set of 10 methods based on simple correlation, mutual infromation and non-linear correlation are available to the user.

View details »

Network Stability

The procedure computes the network stability indicators given the inference method and the network distance choosen. The method is based on perturbation introduced in the data from a resampling procedure.
